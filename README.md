# 出遊費用計算系統

## 專案簡介

這是一個專為團體出遊設計的費用計算系統，主要用於計算關子嶺2026年度旅遊的各項費用。系統能夠根據不同的參加者類型（同工、家眷、小孩）以及選擇的房型，自動計算出每個人的費用明細與總金額。

## 專案結構

```
Guanziling2026/
├── index.html      # 主要 HTML 結構
├── main.js         # 費用計算邏輯
├── style.css       # 樣式表（包含 RWD 響應式設計）
└── README.md       # 專案說明文件
```

## 功能特色

### 1. 多類型人員計算
- **同工**：基本費 1,000 元 + 房型加價
- **家眷**：車資(800) + 保險(500) + 餐費(300) + 房價
- **小孩(110cm-150cm)**：固定 800 元
- **小孩(110cm以下)**：免費

### 2. 多種房型選擇
- 二人房：加價 300 元（同工）/ 2,500 元（家眷）
- 四人房：無加價（同工）/ 2,200 元（家眷）
- 二人房(含湯屋)：加價 800 元（同工）/ 3,000 元（家眷）
- 四人房(含湯屋)：加價 300 元（同工）/ 25,000 元（家眷）
- 六人房：加價 300 元（同工）/ 2,500 元（家眷）

### 3. 即時費用明細
系統會為每位參加者生成詳細的費用明細表，包含：
- 車資
- 保險
- 餐費
- 房型/加價費用
- 個人小計

### 4. 響應式設計（RWD）
- **桌面版**：左右分欄顯示（表單 + 明細）
- **手機版**：上下排列，表格可橫向滑動
- 針對觸控裝置優化的按鈕和輸入框尺寸

## 使用方式

### 基本操作流程

1. **輸入參加人數**：填寫總參加人數
2. **分類人數**：
   - 輸入同工人數
   - 輸入家眷人數
   - 輸入小孩(110cm-150cm)人數
   - 輸入小孩(110cm以下)人數
3. **選擇房型**：從下拉選單選擇合適的房型
4. **確認固定費用**：
   - 車資：800 元（不可修改）
   - 保險：500 元（不可修改）
   - 餐費：300 元（不可修改）
5. **點擊計算按鈕**：系統會自動計算並顯示結果

### 輸入驗證

- 參加人數必須大於 0
- 同工 + 家眷 + 小孩人數總和必須等於參加人數
- 若驗證失敗，系統會顯示警告訊息

## 技術架構

### HTML (index.html)
- 使用語義化標籤建構表單
- 雙欄式佈局：表單區（form-section）和明細區（detail-section）
- 無障礙設計：使用 label 標籤提升可讀性

### JavaScript (main.js)
- **主要函式**：`calc()` - 執行所有計算邏輯
- **費用計算公式**：
  ```
  同工總費用 = 人數 × (1000 + 房型加價)
  家眷總費用 = 人數 × (800 + 500 + 300 + 房價)
  小孩(110-150)總費用 = 人數 × 800
  小孩(<110)總費用 = 人數 × 0
  ```
- **動態表格生成**：使用 `addDetailRow()` 函式動態產生每個人的費用明細

### CSS (style.css)
- **Flexbox 佈局**：實現響應式雙欄設計
- **卡片設計**：使用陰影和圓角營造層次感
- **RWD 斷點**：768px
  - 桌面版：flex-direction: row
  - 手機版：flex-direction: column
- **表格優化**：手機版可橫向滾動，避免內容溢出

## 計算邏輯說明

### 同工費用計算
```javascript
基本費：1,000 元
房型加價：根據選擇的房型（0-800 元不等）
總費用 = 基本費 + 房型加價
```

### 家眷費用計算
```javascript
基本費 = 車資(800) + 保險(500) + 餐費(300) = 1,600 元
房價：根據選擇的房型（2,200-25,000 元不等）
總費用 = 基本費 + 房價
```

### 小孩費用計算
```javascript
110cm-150cm：固定 800 元
110cm以下：0 元（免費）
```
